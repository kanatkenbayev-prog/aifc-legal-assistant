# МФЦА Правовой Ассистент — AIFC Legal Assistant

Интерактивный AI-ассистент для юридических консультаций по законодательству МФЦА (Международного финансового центра «Астана») и Республики Казахстан.

## Возможности

- Структурированные ответы в формате юридической консультации (7 разделов)
- Веб-поиск актуальных норм на `aifc.kz` и `adilet.zan.kz`
- Предупреждения об изменениях законодательства за последние 12 месяцев
- 8 категорий типовых запросов
- Выбор области права (корпоративное / финансовое / налоговое / трудовое / споры / комплаенс)
- Поддержка русского и английского языков
- История последних 5 запросов

## Запуск

1. Откройте `index.html` через локальный сервер (из-за CORS нельзя открывать двойным кликом):
   ```bash
   npx serve .
   # или
   python3 -m http.server 8080
   ```
2. Введите API-ключ Anthropic (`sk-ant-...`) в поле на странице
3. Задайте вопрос

## Деплой на GitHub Pages

1. В настройках репозитория → **Pages** → Source: `main` / `/ (root)`
2. Сайт будет доступен по адресу `https://<username>.github.io/aifc-legal-assistant/`

> **Важно:** API-ключ вводится пользователем в браузере и не хранится на сервере.

## Источники права

1. Нормативные акты МФЦА — https://aifc.kz/legal-framework/acts/
2. Правила МФЦА — https://aifc.kz/legal-framework/rules/
3. Руководства МФЦА — https://aifc.kz/legal-framework/guidance/
4. Конституционный закон РК «О МФЦА» № 438-V от 7 декабря 2015 года
5. Законодательство РК — https://adilet.zan.kz
6. Акты AFSA, Национального банка РК, АРРФР, AIX
